LLM ニュース

2026年6月17日（水） 過去1週間

まとめ

今週の LLM 業界は「フロンティアモデルを巡る地政学」が前面に出た。米政府が国家安全保障を理由に Anthropic の最上位モデル Claude Fable 5 / Mythos 5 への外国籍ユーザーのアクセスを停止する輸出規制指令を出し、Anthropic は全ユーザー向けに Fable 5 を一時停止した。ほぼ同時に中国の Zhipu が GLM 5.2 を MIT ライセンスで完全オープンソース化し、規制で囲い込む米国と開放で押し出す中国という対照が鮮明になった。実利面では「エージェントの本番投入」が共通テーマで、Anthropic は Claude Code の経済価値を定量化した研究と Managed Agents の本番運用ガイドを、OpenAI は Codex の大型アップデートとリリース前に挙動を予測する Deployment Simulation を公開。動画生成では xAI の Grok Imagine Video 1.5 が API 一般提供に入った。

今週の重要ポイント

米政府が Claude Fable 5 / Mythos 5 に輸出規制 — 全ユーザーで利用停止

米政府が国家安全保障権限を根拠に、米国内外を問わずすべての外国籍ユーザー（Anthropic の外国籍従業員を含む）による Fable 5・Mythos 5 へのアクセスを停止する輸出規制指令を発出。Anthropic は遵守のため両モデルを全顧客向けに停止した。公開直後の最上位モデルが規制で即座に封じられる異例の事態となった。

  • Anthropic — Statement on the US government directive to suspend access to Fable 5 and Mythos 5
  • Al Jazeera — US orders Anthropic to disable AI models for all foreign nationals

Zhipu が GLM 5.2 を MIT で完全オープン化 — 1M コンテキストの長尺エージェント特化

中国の Zhipu (Z.ai) が GLM 5.2 を MIT ライセンスで完全オープンソース化。744B パラメータ（アクティブ 40B）の MoE で、コンテキスト長は前世代の約 5 倍となる 100 万トークン、長尺の計画・実装・デバッグを横断する自律的ソフトウェアエンジニアリングに照準を合わせる。米国が最上位モデルを規制で囲う週に、対照的にフロンティア級モデルを無制限で開放した格好だ。

  • StableLearn — GLM-5.2 Goes Fully Open: 753B Parameters, 1M Context
  • Codersera — GLM 5.2 Release — 1M Context, Coding-First

Anthropic Economic Index、Claude Code の価値増大を定量化

Anthropic が約 40 万件の Claude Code セッション（2025年10月〜2026年4月）を分析し、利用が end-to-end のエージェント的作業へ移行し、タスクあたりの経済価値が約 25% 上昇したと報告。人間が「何を作るか（計画）」を担い Claude が「どう作るか（実行）」を担う分業や、ドメイン専門性が高いほど 1 プロンプトで Claude がこなす仕事量が増えるという傾向を示した。Claude Code の run-rate 収益は 25 億ドルを突破したとされる。

  • Anthropic — The Anthropic Economic Index report: New building blocks for understanding AI use
  • Anthropic (X) — a framework for tracking Claude Code as it scales

OpenAI、リリース前に挙動を予測する Deployment Simulation を公開

OpenAI が、実際の運用データを使ってモデルの挙動をリリース前に予測する手法 Deployment Simulation を発表。旧モデルの応答を取り除いた匿名化済みの会話を新候補モデルで再生成し、本番投入後にどの程度の頻度で望ましくない挙動が出るかを事前に見積もる。約 130 万件の会話を用い、ステートフルなツールを伴うエージェント運用にも拡張した。

  • OpenAI — Predicting model behavior before release by simulating deployment
  • MarkTechPost — OpenAI Deployment Simulation Extends Pre-Deployment Risk Assessment to Agentic Coding

モデルと規制

輸出規制の波紋 — Build Day は Opus 4.8 に切替、Cognition も Fable 5 を撤去

指令は金曜夜に届き、Anthropic は週末にかけて Fable 5 を全面停止。混乱を受けて 5 時間・週次のレート上限を全ユーザーでリセットし、開発者向け Build Day は Fable 5 ではなく Opus 4.8 を使って予定通り開催する形に切り替えた。Devin を提供する Cognition も自社製品から Fable 5 を撤去するなど、エコシステム全体へ即座に波及した。政府書簡は具体的な安全保障上の懸念を明示しておらず、Anthropic の理解では Fable 5 の「ジェイルブレイク」手法が把握されたことが背景にあるとされる。Fable 5 を巡るアクセス制限は週末を過ぎても継続し、Anthropic とホワイトハウスの対立が長引いている。

  • Fortune — Anthropic disables Fable and Mythos AI models following U.S. government export ban
  • 9to5Mac — Anthropic pulls Claude Mythos 5 and Claude Fable 5 following US government directive
  • Claude Developers (X) — Suspending access to Claude Fable 5; other models remain available

オープンウェイトの存在感 — GLM 5.2 が Notion など製品に即組込み

GLM 5.2 は公開直後から実プロダクトへの統合が進み、Notion が Baseten 経由で GLM 5.2 を「長尺タスク向けの新しいオープンウェイトモデル」として早速採用した。MIT ライセンスで重みが Hugging Face / ModelScope から自由に入手できることが、西側プロダクトへの取り込みハードルを下げている。米国が最上位モデルを規制で締める一方、オープンウェイトの中国製モデルが現場の選択肢として広がる構図が浮かぶ。ベンチマーク値は公開時点で未提示で、性能の主張は今後の検証待ちである。

  • Notion (X) — New open-weight model: GLM 5.2 is now available in Notion (via Baseten)
  • Pasquale Pillitteri — GLM-5.2 Open Source: Zhipu's Answer to the US AI Block

エージェント基盤・本番投入

Claude Managed Agents、本番投入の知見と自前サンドボックス対応

Anthropic の Applied AI チームが、エージェントを本番に乗せる際の課題（認証情報、サンドボックス化、可観測性など）と Managed Agents がそれをどう解くかをまとめたブログを公開。あわせて、利用者が制御するサンドボックス上でエージェントを動かす構成を可能にし、blaxel・e2b・Google Cloud・Namespace など各プロバイダ向けのガイドを追加した。エージェントが長時間動き多くのコードを実行・操作するほど、自前ハーネスの維持コストが重くなるという、運用主体の関心への回答という位置づけだ。

  • Claude Developers (X) — How do teams get agents into production? (Applied AI blog)
  • ChatForest — Claude Managed Agents Now Has Cron Scheduling and Vault Credentials

Claude Agent SDK の別枠クレジット移行は延期

Claude Agent SDK とヘッドレスモード（claude -p）の利用を別枠クレジットへ切り替える予定が、当初告知から延期されたことがメールで通知された。ハーネスを自前で組む開発者の課金体系に直結する変更で、Managed Agents への誘導と合わせて、エージェント運用の「どこにコストを置くか」を巡る設計判断が業界の論点になりつつある。

  • oikon (X) — Agent SDK・ヘッドレスモードの別枠クレジット化が延期

エージェント・開発ツール

OpenAI Codex 大型更新 — レート上限のバンキングとデスクトップ Computer Use

OpenAI が Codex を相次いで強化した。Plus / Pro 向けにレート上限リセットを「貯めて好きなタイミングで使う」バンキング機能を投入（6/11〜24 は紹介で追加リセット獲得）。さらに macOS / Windows でのデスクトップ Computer Use、バックグラウンドのブラウザ操作向け Chrome 拡張、ワークフローや技術スタックを覚える Memories を追加。CLI 側でも /usage のトークン統計や Claude Code からの設定・チャット /import など、競合からの移行を意識した機能が並んだ。

  • OpenAI — Codex Changelog
  • Crypto Briefing — OpenAI introduces free banked reset system for Codex users

Claude Code は週内に 2.1.177〜2.1.179 を連続リリース

Claude Code が一週間で 2.1.177〜2.1.179 を立て続けに公開。ripgrep ベースの Grep をツール説明で推奨してコード検索を高速化したほか、権限ルールでのツール指定、ストリーム中断時の部分応答の保持、サンドボックスの denyRead / allow 制御などを順次追加した。エージェント型コーディング環境の細かな運用品質を詰める動きが続いている。

  • Claude Code Log (X) — Claude Code 2.1.179 highlights
  • Claude Code Log (X) — Claude Code 2.1.178 highlights (ripgrep Grep, permission rules)

OpenAI、リアルタイム音声の次世代「GPT-Realtime-2」を示唆

OpenAI の Greg Brockman 氏が「GPT-Realtime-2 は何か新しいものだ」とデモ付きで投稿し、リアルタイム音声・対話モデルの次世代が近いことを示唆した。エージェントの「目と手」（Computer Use）に続き、「耳と口」にあたるリアルタイム対話の刷新が、対話型エージェントの体験を一段押し上げる可能性がある。

  • Greg Brockman (X) — GPT-Realtime-2 is something new

メディア生成・コンシューマー

xAI、Grok Imagine Video 1.5 を API 一般提供 — 720p・物理表現を強化

xAI が画像から動画を生成する Grok Imagine Video 1.5 を API で一般提供開始。リアリズムと物理表現を高め生成速度も向上させ、出力は最大 720p、音声生成も標準で含む。コンシューマー向けには待ち時間を大幅に短縮した Video 1.5 Fast も展開した。動画生成は Image-to-Video のリーダーボードでも上位を争う競争領域となっており、xAI が API 提供で開発者の取り込みを狙う。

  • xAI — Grok Imagine 1.5 Preview
  • xAI (X) — Imagine Video 1.5 is generally available in our API

Google、NotebookLM を AI Ultra で全面展開・Gemini に「Daily Brief」

Google は、より高機能になった NotebookLM 体験を Google AI Ultra 契約者向けに世界で 100% 展開したと発表。チャートや画像、スプレッドシートなどの生成に対応する。Gemini アプリでは、起床前に一日を整理する朝のエージェント「Daily Brief」を提供開始するなど、コンシューマー向けの常時稼働エージェント機能の拡充が進む。

  • NotebookLM (X) — More powerful NotebookLM now 100% rolled out to Google AI Ultra
  • Gemini App (X) — Daily Brief, your new morning AI agent
