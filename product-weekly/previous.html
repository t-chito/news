まとめ

今週最大のニュースは Claude Fable 5（Mythos-class）の一般公開（6/9）。Claude Code v2.1.170 からすぐに利用可能になり、Cognition の FrontierCode eval で #1 を獲得した。Claude Code 自体は --safe-mode フラグ追加（v2.1.169）、Dynamic Workflows の GA 化、Apple Foundation Models 対応と週を通じてアップデートが続いた。Anthropic は内部データとして「エンジニアのコード生産性が 2021–2025 比で 8 倍」を公開し、再帰的自己改善への布石として注目を集めた。他プロダクトでは Devin が企業向け $10M 生産性保証と FrontierCode eval を投入、NotebookLM が Gemini 3.5 搭載の大型アップグレードを発表した。

Claude Code

Claude Fable 5 が統合され、チームの働き方が変わった（v2.1.170）

6 月 9 日、Claude Code v2.1.170 がリリースされ、Fable 5 が利用可能になった。CLI 上では /model claude-fable-5 で切り替えられる。Fable 5 はコンテキストウィンドウ 1M トークン、最大 128k 出力トークン対応のモデルで、ソフトウェアエンジニアリング・科学研究・ビジョンで最高水準のベンチマーク結果を示している。

Claude Devs のアカウントは「チームの働き方が変わった。以前は Claude が正しく作業したかを確認していたが、今は正しい仕事をしているかを確認するようになった」と発信。バグフィックスは v2.1.170 で「VS Code 統合ターミナル等から起動したセッションでトランスクリプトが保存されない問題」を修正している。

なおサイバーセキュリティ・生物・化学・蒸留関連のリクエストには安全分類器が働き、Claude Opus 4.8 にフォールバックする仕組みが入っている（発生率は平均セッションの 5% 未満）。LLM 開発領域でのフラグリクエストも同様に Opus 4.8 へ切り替わるよう、今週中に順次ロールアウト中。

  • @ClaudeDevs: Claude Fable 5 is here — getting started in Claude Code
  • Anthropic: Claude Fable 5 and Claude Mythos 5

Dynamic Workflows が GA——数百並列サブエージェントでコードベース全体マイグレーションが対象に

Claude Code における Dynamic Workflows が正式 GA になった。Claude が 10〜100 以上の並列サブエージェントを自律的にオーケストレーションし、数十万行規模のコードベースマイグレーションをキックオフからマージまで実行する。既存のテストスイートが品質バーとして使われる。Code with Claude Tokyo（6/11）でも発表された主要機能のひとつ。

  • Claude Blog: Introducing dynamic workflows in Claude Code
  • @claudeai: dynamic workflows in Claude Code are generally available

v2.1.169: --safe-mode と /cd コマンドが追加

6 月 8 日リリースの v2.1.169 では --safe-mode フラグ（環境変数 CLAUDE_CODE_SAFE_MODE も可）が追加された。CLAUDE.md、プラグイン、スキル、フック、MCP サーバーをすべて無効にした状態で起動するデバッグ用モード。カスタマイズが絡んだ問題の切り分けに使える。あわせて /cd コマンドも追加された（30 件の変更を含む大型リリース）。

バグ修正では JetBrains IDE のフリッカー、WezTerm/Ghostty での Shift+非 ASCII 文字が入力できない問題、Windows での PowerShell コマンド検証ハング、macOS でのゾンビプロセス CPU 100% 消費をそれぞれ修正。

  • @ClaudeCodeLog: Claude Code 2.1.169 has been released
  • GitHub Releases: anthropics/claude-code

Apple Foundation Models フレームワークから Claude を呼べるように

Apple の Foundation Models フレームワーク向けに、Claude を呼び出す Swift パッケージが公開された。iOS 27・iPadOS 27・macOS 27・visionOS 27・watchOS 27 に対応し、複雑な多段推論やコード生成・長文コンテキスト処理を Apple デバイス上のアプリから呼び出せる。

Anthropic 周辺の動き

「エンジニアのコード生産性が 8 倍に」——自己改善への布石として反響

6 月 4 日、Anthropic が内部データを公開。同社エンジニアは 2021〜2025 年比で四半期あたり 8 倍のコード量を出荷しており、難易度の高いオープンエンドな問題での Claude の成功率は 6 か月で 26% → 76% に跳ね上がった（50 pt 増）。Claude が AI 開発自体を加速させる「再帰的自己改善」の可能性がある経路だとして、25 万インプレッションを超える注目を集めた。

Dario Amodei は 6 月 10 日に新論文「Policy on the AI Exponential」を公開。「AI の進化速度は政策立案プロセスが想定していたより遥かに速い」として、業界と政策立案者への提言を行った。

  • @AnthropicAI: AI is accelerating AI development
  • @DarioAmodei: Policy on the AI Exponential

その他プロダクトの動向

Devin: FrontierCode eval で「マージ可能かどうか」を基準に、$10M 生産性保証も

Cognition が 6 月 8 日に FrontierCode を発表。既存の eval は「動くかどうか」を見るが、FrontierCode は正確性・テスト品質・スコープ規律・コードスタイル・リポジトリ標準準拠と、実際にマージできる品質かどうかを評価する。36 の主要 OSS リポジトリのメンテナと協力して設計されており、タスクあたり 40 時間以上の制作コストがかかっている。Claude Fable 5 は FrontierCode で現時点の #1 を獲得した。

あわせて AI Productivity Guarantee も発表。Devin が支払額に見合う生産性を提供できなければ Cognition が最大 $10M までカバーするという企業向け SLA。タスクごとに AI 推定エンジニア工数を算出・検証する仕組みが裏側にある。

  • Cognition: Introducing FrontierCode
  • Cognition: AI Productivity Guarantee
  • @cognition: Claude Fable 5 in Devin — #1 on FrontierCode

NotebookLM が大型アップグレード——Gemini 3.5 + Antigravity でアジェンティックに

6 月 8 日、NotebookLM が大型アップグレードを発表。Gemini 3.5 と Antigravity で動作し、複雑なマルチステップ調査をチャット内でエージェント的に実行できるようになった。出力フォーマットも拡充され、ソース帰属（Source Attribution）機能も新たに追加。どのプロンプト＋ソースの組み合わせでアーティファクトが作られたかを確認・再現できる。Google AI Ultra および Workspace の AI 拡張アクセス顧客から順次展開中。

  • @NotebookLM: Introducing a more powerful NotebookLM
  • @NotebookLM: Source Attribution launch

ChatGPT のメモリが「常時サマリー型」に進化

6 月 4 日、OpenAI が ChatGPT のメモリシステムをリニューアル。従来の「保存された記憶」だけでなく、会話間で重要な文脈を自動的に保持・更新する新アーキテクチャに移行した。ユーザーはメモリサマリーで保持内容を確認・削除でき、旧来の saved memories に戻すことも可能。段階的にロールアウト中。

  • @OpenAI: New more capable memory system in ChatGPT

Gemini Live で画像作成・編集が可能に、Canvas もリリース

6 月 5 日、Gemini Live でリアルタイム画像作成・編集が解禁。部屋のインテリア検討やミームのその場生成などが会話中にできるようになった。また Gemini 3.5 Flash を使ったコードワンショット生成デモ（Canvas）が話題に。参照画像からライティングを解析しインタラクティブな 3D ビジュアライザを 1 発で生成した動画が 480 万インプレッションを超えた。macOS アプリでは両 Command ⌘ キー同時押しでアクティブウィンドウをチャットに添付できる画面認識機能も追加。

  • @GeminiApp: Create and edit images directly in Gemini Live
  • @GeminiApp: Gemini 3.5 Flash Canvas demo

Grok Voice Think Fast 1.0——EVA-Bench で精度と体験のパレート最前線

xAI が Grok Voice Think Fast 1.0 をリリース。人間に近いタイミング・トーン・温かみを持ち、EVA-Bench では「精度を落とさずに体験を犠牲にしない」パレートフロンティアで競合を上回ったと発表。価格は競合の一部とされており、音声エージェント分野での競争が本格化している。
