まとめ

今週の週間トレンドは「AI エージェントが何を作るか」へ重心が移った。首位は登場直後に週間 +1.5 万★を集めた calesthio/OpenMontage で、コーディングエージェントを動画制作スタジオに変える「エージェント型の制作システム」だ。先週まで上位を占めた「スキル集」型は AI セキュリティスキル集や system_prompts_leaks として健在だが、主役は deer-flow・withastro/flue・orca といったエージェントを動かす土台（フレームワーク／ハーネス）と、voicebox・OpenCut のような AI メディア生成ツールに移っている。worldmonitor や daily_stock_analysis のように LLM で情報を統合する「インテリジェンス・ダッシュボード」も複数ランクインした。AI 以外では penpot・Stirling-PDF・iroh といった定番 OSS が安定して新規スターを集めている。

AI エージェント・フレームワーク／ハーネス

bytedance/deer-flow — 長尺タスクをこなす SuperAgent ハーネス（週間 +3,242★ / 累計 7.5万★）

ByteDance による、「調査し、コードを書き、創作する」長期ホライズン型 SuperAgent のオープンソースハーネス。複雑なタスクを文脈の独立したサブエージェントに分解して並列実行し、サンドボックス（Docker または local）でファイル操作やコード実行を行う。研究・レポート生成・スライド作成といった機能をモジュール化したスキルで拡張でき、セッションを跨いでユーザーの好みや文脈を学習する長期メモリも持つ。

Telegram・Slack・Feishu・WeChat・DingTalk などメッセージング基盤と接続でき、Web 検索・bash 実行・MCP サーバー連携をツールとして備える。LangChain／LangGraph 上に構築され、複数の LLM プロバイダに対応する。

  • bytedance/deer-flow

withastro/flue — Astro 発、TypeScript のサンドボックスエージェントフレームワーク（週間 +1,327★）

Astro チームが手がける「サンドボックスエージェントフレームワーク」。README は「もう一つの SDK ではない」と謳い、TypeScript のプログラマブルなハーネスとして自律エージェントと AI ワークフローを組むことを掲げる。ゴールに向けて自律的に動きつつセッションの連続性を保つエージェント、推論を導く構造化ワークフロー、ツールやファイル変更を隔離する安全なサンドボックス、失敗から復帰する持続的実行（durable execution）を中核機能とする。

委譲用のサブエージェント、型付きツール、再利用可能なスキル、MCP サーバー連携に加え、OpenTelemetry・Braintrust・Sentry での可観測性、Slack・Teams・Discord・GitHub のイベントチャネルを備える。Node.js・Cloudflare Workers・GitHub Actions・GitLab CI/CD などへデプロイできる。Web フレームワークで知られる Astro が、エージェント基盤へ軸足を広げたことを示す動きだ。

  • withastro/flue

stablyai/orca — 並列エージェントの「艦隊」を操る ADE（週間 +1,949★）

複数のコーディングエージェントを並列で走らせるための ADE（Agentic Development Environment）。README は「並列エージェントの艦隊（fleet）で作業するための ADE。自分のサブスクで任意のコーディングエージェントを動かせる。デスクトップとモバイルで利用可能」と説明する。1 つのプロンプトから複数エージェントを隔離された worktree で同時に走らせ、結果を比較してベストをレビュー・マージできる。

遠隔から監視・操舵するモバイルアプリ、WebGL レンダリングの高機能ターミナル、UI 要素をクリックして HTML/CSS をプロンプトに注入する Design Mode、GitHub・Linear 連携、リモートサーバー上でエージェントを動かす SSH worktree、注釈可能な AI diff などを備える。Claude Code・Codex・OpenCode など 20 以上のエージェントに対応する。

  • stablyai/orca

Panniantong/Agent-Reach — API 不要でネット全体に手を伸ばす CLI（週間 +6,830★ / 累計 4.1万★）

「AI エージェントにインターネット全体を見る目を与える」CLI。Twitter/X・Reddit・YouTube・GitHub・Bilibili・小紅書（XiaoHongShu）・LinkedIn・RSS・一般 Web ページを、有料 API なしで読み取り・検索できる。各プラットフォームに主系・予備のアクセス手段を持ち、一方が失敗すると自動で切り替わる多重バックエンド構成が特徴で、先週に続き今週も週間 6,000 超のスターを集めて累計 4.1 万に達した。Claude Code・Cursor・Windsurf などシェルコマンドを扱える任意のエージェントから使え、agent-reach doctor による各経路のヘルスチェックも備える。

  • Panniantong/Agent-Reach

AI メディア生成（動画・音声）

解説動画・トーキングヘッド・シネマティックな予告編・ドキュメンタリー・アニメーション・ポッドキャストなど 12 の制作パイプラインを持ち、動画／画像生成・音声合成・音楽・ミキシング・字幕の 52 ツールを束ねる。Archive.org・NASA・Wikimedia Commons・Pexels・Unsplash などの無料アーカイブから実写ドキュメンタリーを作れるほか、脚本前のライブ Web 調査、Kling・Runway・Google Veo・ローカル GPU モデルなど 14 以上の動画生成プロバイダ対応、参照動画の分析（YouTube Short を貼ると差別化した派生案を出す）、品質ゲートとコスト管理を備える。ライセンスは AGPLv3。

AI × インテリジェンス・分析

google-research/timesfm — Google の時系列予測基盤モデル（週間 +3,040★ / 累計 2.5万★）

Google Research による、時系列予測のための事前学習済み基盤モデル（Time Series Foundation Model）。予測に特化したデコーダのみの Transformer 構造で、最大 16,000 の文脈長（過去データ点）を受け、最大 1,000 ステップ先までを予測する。不確実性を表す分位点（quantile）予測にも対応し、PyTorch と Flax の両実装が提供される。

BigQuery ML 連携によるエンタープライズ予測、Google スプレッドシートでの予測、Vertex Model Garden 経由のスケーラブルな推論といった導入経路を持つ。金融・気象・株式など領域を問わない汎用の時系列分析に使える。ライセンスは Apache 2.0。

n0-computer/iroh — IP ではなく公開鍵でつなぐ P2P ネットワーク基盤（週間 +886★）
